# Architecture Diagramer

This skill enables creating comprehensive architecture, application, flow diagrams and any other
diagram included in the different documentation markdowns using Mermaid diagrams whenever it makes
sense.

## When to Use This Skill

Use this skill when you need to:

1. Create architecture diagrams for system components
1. Design application flowcharts and workflows
1. Visualize data flows and interactions
1. Document system architecture in markdown files
1. Generate diagrams for technical documentation

## DOs

- Use Mermaid syntax for diagrams
- Use C4 model for architecture diagrams
- Ensure diagrams are clear and easy to understand
- Include a copy of the diagram with a comprehensive name under [docs/diagrams](/docs/diagrams/)
