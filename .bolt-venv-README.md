# Bolt Framework - Python Environment Notes

## Purpose

This directory contains the Python virtual environment for Bolt Framework scripts.
Created automatically by `.aurora/scripts/powershell/Bootstrap-Python.ps1`.

## Contents

- **bin/** (Linux/macOS) or **Scripts/** (Windows) - Python executables
- **Lib/site-packages/** - Installed packages (anthropic, pyyaml)

## Regenerate

If this environment becomes corrupted:

```powershell
# Windows
Remove-Item -Recurse -Force .bolt-venv
.\.aurora\scripts\powershell\Bootstrap-Python.ps1

# Linux/macOS
rm -rf .bolt-venv
source .aurora/scripts/bash/bootstrap-python.sh
```

## Do NOT Commit

This directory is gitignored. Each developer creates their own local environment.
